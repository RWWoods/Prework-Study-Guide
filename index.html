HTML

  • The head element contains information about the webpage
  • The body element represents visible content shown to the user
  • Alt describes content for accessibility and clarity

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we a re currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • cd in the command line will take you to a specified directory, or home if blank
  • pwd in command line prints current location
  • ls in command line lists contents of present location

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • In JavaScript, arrays start index at 0, not 1.
  • Use var to declare a variable, then the contents of the variable follow the = sign
